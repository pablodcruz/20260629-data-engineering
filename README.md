# Data Engineering Learning Hub

This repository collects review notes, hands-on labs, project planning docs, Docker examples, and Python coding challenges for a practical data engineering training path.

The focus is on building confidence with the tools and habits data engineers use every day: reading data, writing clean Python, working with containers, understanding streaming concepts, debugging systems, and explaining technical choices clearly.

## Interactive Study App

Open [study/index.html](study/index.html) for randomized Week 6–8 practice, exams, flashcards, smart review, and a locally saved mastery dashboard.

The question bank covers Spark, Spark SQL, Kafka, and Airflow. See [study/README.md](study/README.md) for usage and question-authoring instructions.
