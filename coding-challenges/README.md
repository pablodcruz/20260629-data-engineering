# Coding Challenges

This folder contains short Python coding challenges for a coding assessment.
The goal is to evaluate programming fundamentals, logical thinking, and coding approach.

These exercises stay within beginner-friendly Python concepts:

* Strings
* Loops
* Conditionals
* Lists and dictionaries
* Functions
* Classes
* File reading
* Exception handling

## Files

| File | Purpose |
| --- | --- |
| `challenges.md` | Full challenge prompts, examples, and expected behavior |
| `starter_code.py` | Empty functions and classes to complete |
| `sample_tests.py` | Simple assert-based checks that run locally |
| `sample_data/scores.txt` | Small input file for the file-handling challenge |

## Running the Sample Tests

From this folder, run:

```bash
python sample_tests.py
```

The tests are intentionally simple.
