# Project Phase 1

## Project Title

**StreamFlow Phase 1 - Containerized Stream Processing Platform**

## Project Description

Design and implement a fully containerized, end-to-end streaming data platform using Apache Kafka, PySpark, Apache Airflow, and Docker. The platform ingests real-time event streams, processes them through Spark batch jobs, and orchestrates workflows using Airflow DAGs within a local distributed environment.

## Objective / Tools Used

* Apache Kafka
* PySpark DataFrames / SparkSQL
* Apache Airflow
* Docker / Docker Compose
* Python
* Distributed Data Processing Concepts
* Workflow Orchestration

## Weeks During Training

Weeks 1-3 (6-8 total)

## Project Type

Group Project

## Outcomes

* Understand distributed data processing fundamentals.
* Build event-driven data pipelines using Kafka.
* Learn Spark DataFrame transformations and Spark execution concepts.
* Orchestrate workflows using Apache Airflow.
* Gain experience with containerized development environments.
* Learn micro-batch architecture patterns and pipeline parameterization.
