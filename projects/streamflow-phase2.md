
# Project Phase 2

## Project Title

**StreamFlow Phase 2 - Enterprise Analytics Pipeline**

## Project Description

Extend the StreamFlow platform by loading processed event data into Snowflake using Medallion Architecture (Bronze/Silver/Gold), building dimensional models and fact tables, and delivering business insights through interactive Power BI dashboards with DAX measures.

## Objective / Tools Used

* Snowflake
* Medallion Architecture
* SQL Transformations
* Apache Airflow
* Power BI Desktop
* DAX (Data Analysis Expressions)
* Star Schema / Dimensional Modeling
* Data Warehousing Concepts

## Weeks During Training

Weeks 4-6 (9-11 total)

## Project Type

Group Project

## Outcomes

* Understand modern analytics and data warehousing architecture.
* Implement Bronze, Silver, and Gold data layers.
* Design dimensional models and star schemas.
* Build analytics-ready warehouse tables and KPIs.
* Create interactive dashboards and BI reports.
* Gain hands-on experience with Snowflake and cloud data warehousing concepts.
* Understand how raw event data is transformed into business insights.
