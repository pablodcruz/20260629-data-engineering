# Syllabus

## Overview

The Data Engineering cohort extends the foundation track into pipeline design, orchestration, data modeling, and production-minded data workflows.

## Draft Weekly Arc

1. Data engineering orientation and environment setup
2. Batch ingestion patterns
3. Data validation and quality checks
4. PostgreSQL modeling for analytics
5. Pipeline orchestration concepts
6. Incremental loading and failure recovery
7. Warehouse-style transformations
8. Final project build and demos

Adjust the pacing based on the cohort's readiness and project progress.
