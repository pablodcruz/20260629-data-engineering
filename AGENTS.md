# AGENTS.md

This repo is for the Data Engineering cohort that follows the 20260523 Data Foundations course.

When helping in this repo:

- Keep examples practical and cohort-friendly.
- Prefer clear, teachable structure over clever abstractions.
- Assume students have completed Python, Git, PostgreSQL, and basic ingestion work from the foundation track.
- Build toward production data engineering habits: repeatability, observability, data quality, orchestration, and documentation.
- Avoid committing secrets, `.env` files, virtual environments, generated datasets, or machine-specific files.
